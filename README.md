Русскоязычная версия сайта посвящённого веб-фреймворку Sinatra
================================================================

### Работа с сайтом

    git clone git://github.com/sinatraru/sinatraru.github.com.git
    cd sinatraru.github.com

Устанавливаем зависимости

    gem install jekyll

И запускаем встроенный сервер

    rake server
